== Installation ==

1. Copy Leniy-Tweaks.zip to your Wordpress plugins directory, usually `wp-content/plugins/` and unzip the file.  It will create a `wp-content/plugins/Leniy-Tweaks/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
